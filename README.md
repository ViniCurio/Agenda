# API Agenda

Um projeto simples em Typescript, API para uma web agenda que permite gerar, consultar e apagar agendamentos.

## Funcionalidades

- Gera um agendamento, onde id é definido automaticamente
- Permite colocar ou não observações sobre o agendamento
- Exibe a agenda completa
- Apaga agendamentos por ID

## Como usar

  Assim que o projeto estiver completo vai ter um link direto disponibilizado aqui

## Requisitos

- Navegador moderno (Chrome, Firefox, Edge, Safari)

## Observações

- O projeto utiliza um mongo DB para armazenar os agendamentos, pois futuramente vai guardar histórico dos agendamentos com login e senha para os usuários da agenda.
- Ainda não esta finalizado, em breve vai ser criado o frontend da aplicação, para teste e uso.
